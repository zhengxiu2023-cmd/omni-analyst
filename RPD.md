📄 产品需求文档 (PRD) - The Omni-Analyst v7.5

1. 产品定位

一个运行在本地高性能工作站（如配备 4060 Ti）上的 CLI 情报终端，专门用于 A 股市场的先期数据挖掘、逻辑过滤与尽调底稿（PDF）的自动化打包。

2. 核心痛点与解决思路

痛点: 原代码所有逻辑耦合在一个上帝类中，网络请求极易崩溃，特定公告（如调研纪要、竞对年报）下载逻辑断链。

解决: 实施“化整为零”的架构设计，网络层、解析层、业务层彻底解耦。

3. 核心功能模块 (Core Features)

模块 A：防弹网络层 (Military-Grade Network Engine)

提供统一的带有指数退避算法（Exponentiala Backoff）的请求方法。

提供专门应对几十兆 PDF 文件的分块流式（Chunk Streaming）下载器，防内存溢出。

模块 B：先知矩阵 (Oracle Radar)

数据嗅探: 穿透龙虎榜、机构调研、全球产业快讯、新闻联播。

全网社交流量监控: 引入基于外部 API 的社交媒体热榜数据，监控跨平台流量流动与病毒系数，作为全新的另类情报源。

双擎提纯:

物理引擎：基于正则词库（颠覆、断供、暴涨）进行初筛。

神经引擎：调用本地 Ollama (Qwen) 判断情报是否具备“超景气与奇点”特征，输出精准星级。

模块 C：底稿抽吸泵 (Cninfo Extractor)

基于巨潮资讯网 API，根据代码智能适配深/沪市。

强制提取目标股：最近两年年报、最新季报、所有关联的投资者问答/调研纪要。

强制提取竞对股：自动根据行业和市值匹配最强对手盘，提取其年报与最新季报。

竞对财报获取：新增季报抓取模块，能获取目标公司及其同板块竞争对手的近 8 期核心财报数据，为云端 AI 提供交叉验证底料。

模块 D：风控面板组装引擎 (Risk Panel Builder)

计算前复权 3 年历史极值、当前分位、最新 PE/PB/换手率。

执行硬核物理熔断判定（死亡换手率、ST 暴雷、极端透支）。

整合上述所有信息与大模型提取的 PDF 增量信息，生成格式化的 00_参数面板_发给AI.md，支持增量写入而不覆盖用户的历史修改。
